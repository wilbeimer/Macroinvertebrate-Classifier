# Macroinvertebrate-Classifier
An interdisciplinary project at Shenandoah University combining Computer Science and Environmental Science to classify aquatic macroinvertebrates from images using machine learning for water quality assessment and ecological research.

## Data Information
327 raw images

## Modules

- [nb](docs/nb.md) — image chunking, feature extraction, NB prediction utilities
- [nb_helper_functions](docs/nb_helper_functions.md) — batch prediction pipeline, labeling, visualization
